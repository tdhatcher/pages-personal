## What is Copilot CLI?

GitHub Copilot CLI is a terminal assistant that brings AI-powered coding assistance directly to your command line. It allows you to build, debug, and understand code through natural language conversations.

## Key Features

- **Terminal-native**: Work with Copilot directly in your CLI
- **GitHub integration**: Access repositories, issues, and pull requests
- **Agentic capabilities**: Plan and execute complex tasks
- **Local development**: Run locally and test before deployment

## Getting Started

To get started with Copilot CLI:

1. **Install** the CLI via npm, Homebrew, or the install script
2. **Authenticate** with your GitHub account
3. **Select a model** using `/model` command
4. **Start chatting** with natural language prompts

## Models Available

- **Claude Haiku** - Fast and cost-effective for simple tasks
- **Claude Sonnet** - Balanced for planning and complex analysis
- **Claude Opus** - Premium reasoning for architectural decisions

More information available at [docs.github.com/copilot/cli](https://docs.github.com/copilot/how-tos/use-copilot-agents/use-copilot-cli)
