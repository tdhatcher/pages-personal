## Blogs

- [**Andrew Lock - .NET Escapades**](https://andrewlock.net) — Deep ASP.NET Core internals. The most thorough .NET blog I know.
- [**Steven Giesel**](https://steven-giesel.com) — Practical .NET tips, performance, and architecture.
- [**Anthony Simmon**](https://anthonysimmon.com) — .NET and Azure content from a Workleap staff developer.
- [**Scott Hanselman**](https://www.hanselman.com/blog/) — Microsoft legend. Broad range from .NET to productivity to accessibility.
- [**Khalid Abuhakmeh**](https://khalidabuhakmeh.com) — JetBrains dev advocate. .NET, open source, creative tech writing.

## Newsletters

- [**The .NET Weekly**](https://www.dotnetweekly.com) — Curated .NET links every week. Good signal-to-noise ratio.
- [**TLDR**](https://tldr.tech) — Daily tech digest. The dev edition is worth the 5 minutes.
- [**ByteByteGo**](https://blog.bytebytego.com) — System design concepts with great diagrams. Alex Xu's newsletter.
- [**The Pragmatic Engineer**](https://newsletter.pragmaticengineer.com) — Gergely Orosz on engineering culture, big tech, and career. The best paid newsletter in the industry.
